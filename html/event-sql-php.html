Náročnost: 2


Nevyhnutnou súčasťou webovej aplikácie je databáza pre uchovávanie údajov. Pri väčšine dnešných aplikácii ide o relačné databázy, pre prácu s ktorými developeri používajú jazyk SQL. Tento prístup však nekorešponduje s objektovo orientovaným prístupom, ktorý je dnes samozrejmosťou. Preto vznikla technika známa ako Objektovo Relačné Mapovanie, ktorá dnes nahradzuje štandarný prístup pomocou SQL požiadaviek.
